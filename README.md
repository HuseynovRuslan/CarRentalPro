# Car Rental Mobile App
Built with React Native & Expo Router.

## Project Details
This application allows users to discover, favorite, and book available cars for rental. The project implements modern UI practices with dark/light mode and customized state management.
Designed by following a given Figma mockup, carefully translated into pixel-perfect components.

### Highlights
- Configured Expo Router for tabs and nested routes
- Used Zustand for global state (`bookingStore`, `carStore`, etc.)
- Dynamic theming with custom React Context (`themeContext`)
- Form validation implemented using `react-hook-form` and `zod`

## Quick Start
1. Install everything:
```bash
npm install
```

2. Run Expo:
```bash
npx expo start -c
```
